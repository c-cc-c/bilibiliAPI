### 使用方法

``` url
127.0.0.1?mid=2033
```
其中<mark><b>2033</b></mark>为你的mid, 将得到<b>姓名</b>、<b>个性签名</b>、<b>头像</b>


携带参数all
``` url
127.0.0.1?mid=2033&all
```

则得到<b>全部输出</b>
