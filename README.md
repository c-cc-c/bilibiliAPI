# 简单接口

## 使用例子：

``` url
127.0.0.1?mid=2033
```
其中<code>2033</code>为你的mid

以上将得到<b>姓名</b>、<b>个性签名</b>、<b>头像</b>

如:
``` json
   {
        "name":"初音MIKU",
        "sign":"",
        "avatar":"http:\/\/i0.hdslb.com\/bfs\/face\/member\/noface.jpg"
    }
```

携带参数如：
``` url
127.0.0.1?mid=2033&all
```

则得到全部输出
